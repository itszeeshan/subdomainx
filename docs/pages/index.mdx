# Overview

All-in-one subdomain enumeration tool that combines the power of multiple tools into a single, efficient command-line interface.

## Features

- **Multiple Tools**: Integrates subfinder, amass, findomain, and more for comprehensive enumeration
- **API Integration**: SecurityTrails, VirusTotal, and Censys APIs for enhanced discovery
- **High Performance**: Concurrent execution with configurable threads and rate limiting
- **HTTP Scanning**: Built-in httpx integration for web service discovery and analysis
- **Flexible Output**: Generate reports in JSON, TXT, or beautiful HTML formats
- **Easy to Use**: Simple CLI interface with sensible defaults and optional configuration
- **Tool Management**: Automatic tool detection and installation guidance for missing dependencies
- **Custom Wordlists**: Support for custom wordlists in brute-forcing tools
- **Progress Reporting**: Real-time progress bars with ETA for long-running scans
- **Resource Management**: Memory and CPU monitoring with optimization recommendations

## Quick Start

### 1. Install

```bash
go install github.com/itszeeshan/subdomainx@latest
```

### 2. Run scan

**Single domain:**

```bash
subdomainx --subfinder --httpx example.com
```

**Multiple domains:**

```bash
echo "example.com" > domains.txt
subdomainx --wildcard domains.txt --format html
```

**With API integration:**

```bash
# Set API keys
export SECURITYTRAILS_API_KEY="your_key"
export VIRUSTOTAL_API_KEY="your_key"
export CENSYS_API_ID="your_id"
export CENSYS_SECRET="your_secret"

# Use APIs
subdomainx --securitytrails --virustotal --censys example.com
```

**With custom wordlist:**

```bash
# Use custom wordlist for brute-forcing
subdomainx --wordlist /path/to/wordlist.txt example.com
```

**With progress monitoring:**

```bash
# Enable verbose mode for resource monitoring
subdomainx --verbose --subfinder --amass example.com
```

## What's Included

### Enumeration Tools <a href="/supported-tools#enumeration-tools" className="details-link">details</a>

- **subfinder** - Fast subdomain discovery tool with passive online sources
- **amass** - Comprehensive network reconnaissance and attack surface mapping
- **findomain** - Cross-platform subdomain enumeration tool
- **assetfinder** - Find domains and subdomains potentially related to a given domain
- **sublist3r** - Fast subdomain enumeration tool for penetration testers
- **knockpy** - Python3 tool designed to enumerate subdomains on a target domain
- **dnsrecon** - DNS enumeration and reconnaissance tool
- **fierce** - DNS reconnaissance tool for locating non-contiguous IP space
- **massdns** - High-performance DNS stub resolver
- **altdns** - Generates permutations, alterations and mutations of subdomains
- **waybackurls** - Fetch URLs from Wayback Machine for subdomain discovery
- **linkheader** - Discover subdomains from HTTP Link headers

### API Tools <a href="/supported-tools#api-tools" className="details-link">details</a>

- **SecurityTrails** - Comprehensive subdomain database and historical data
- **VirusTotal** - Security-focused subdomain discovery and threat intelligence
- **Censys** - Internet-wide scanning data for subdomain enumeration

### Scanning Tools <a href="/supported-tools#scanning-tools" className="details-link">details</a>

- **httpx** - Fast and multi-purpose HTTP probe for web services
- **smap** - Port scanner and service discovery tool
